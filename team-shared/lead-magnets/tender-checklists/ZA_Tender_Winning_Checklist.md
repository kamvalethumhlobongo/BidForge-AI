# South Africa — Tender-Winning Checklist
### 10 Things You MUST Have Before Submitting Your Next SA Government Bid

**Why 70% of South African tenders are rejected before evaluation:**
Non-compliance with B-BBEE, CSD, or PPPFA requirements.

---

## The Pre-Submission Audit

### 1. B-BBEE Certification and Status Level
- [ ] B-BBEE Certificate current (issued within last 12 months)
- [ ] B-BBEE Level 1-4 preferred; Level 5-8 varies by tender
- [ ] Full scorecard (not just affidavit) for tenders over R10M
- [ ] Sector code compliance (Construction, Financial, etc.)
- [ ] Supplier development commitments documented and verifiable
- [ ] Enterprise development contributions (2% of NPAT on ED if required)

**Pro Tip:** A Level 1 B-BBEE contributor gets 25 points automatically under 80/20 PPPFA scoring. Without a valid certificate, you score ZERO on empowerment.

### 2. CSD (Central Supplier Database) Registration
- [ ] CSD registration active (verify at www.csd.gov.za)
- [ ] SARS Tax Clearance status Compliant (verified through CSD)
- [ ] Correct commodity codes selected
- [ ] Banking details confirmed via CSD
- [ ] SMME / EME / QSE classification correct
- [ ] CSD profile complete with all mandatory fields

**Pro Tip:** Your CSD profile is the FIRST thing evaluators check. If tax clearance shows Not Compliant, your bid won't be opened.

### 3. Tax and Financial Compliance
- [ ] SARS Tax Clearance Certificate valid (PIN-based system)
- [ ] Tax compliance maintained during tender validity period
- [ ] Audited financial statements last 3 years (tenders over R10M)
- [ ] CIDB financial grading (construction tenders)
- [ ] No outstanding tax debts verified with SARS
- [ ] UIF and PAYE compliance up to date

### 4. PPPFA Compliance (Preferential Procurement)
- [ ] PPPFA scoring understood: 80/20 (under R1M) or 90/10 (R1M+)
- [ ] B-BBEE points claimed based on valid certificate level
- [ ] Local content declaration form signed and submitted
- [ ] Preference points claimed correctly with supporting docs
- [ ] Conditional exemptions documented if applicable
- [ ] Joint venture / consortium agreement compliant (if bidding as JV)

### 5. Local Content and Production
- [ ] Local content declaration completed under PPPFA Regulation 8
- [ ] Local content percentage calculated to minimum thresholds
- [ ] Designated sectors identified (40+ sectors exist)
- [ ] Proof of local manufacturing if claiming local content
- [ ] Local content verification agent engaged if required

### 6. Industry-Specific Compliance

**For Security Companies:**
- [ ] PSIRA registration current at correct grade
- [ ] PSIRA individual registration for all security officers
- [ ] Firearm competency if armed response offered
- [ ] SAPS clearance current
- [ ] B-BBEE sector code compliance for security sector

**For Construction Companies:**
- [ ] CIDB grading correct class and value limit for project
- [ ] NHBRC registration if residential building work
- [ ] Construction B-BBEE Sector Code compliance
- [ ] Project Bank Account compliance if specified
- [ ] Contractor registration with relevant provincial departments

**For Logistics and Transport:**
- [ ] Operating Licence valid at correct class for vehicles
- [ ] Cross-border permits if applicable
- [ ] SABS standards compliance for vehicle specifications
- [ ] Dangerous Goods compliance if transporting hazardous materials
- [ ] Transport B-BBEE Sector Code compliance

**For Cleaning and Facilities:**
- [ ] Waste management licence if handling regulated waste
- [ ] COSHH compliance for chemical handling and storage
- [ ] Healthcare facility clearance if servicing medical facilities
- [ ] Employment equity compliance with affirmative action reporting

### 7. Company Registration and Documentation
- [ ] Company registration (CIPC) current and active
- [ ] Valid ID documents for directors and authorised signatories
- [ ] Resolution to bid authorising the submission
- [ ] Power of attorney if signatory is not a director
- [ ] Companies Act compliance with annual returns up to date

### 8. Tender-Specific Requirements
- [ ] Tender document fully completed with no blank spaces
- [ ] All schedules signed and initialled on every page if required
- [ ] Declarations completed (conflict of interest, no collusion)
- [ ] Technical specification compliance stated
- [ ] Sample or prototype requirement met if applicable
- [ ] Site visit attended if compulsory
- [ ] Briefing session attended if compulsory

### 9. Pricing and Commercials
- [ ] Pricing schedule completed in correct format with no alterations
- [ ] VAT registration valid
- [ ] Price validity period stated (minimum 90 days standard)
- [ ] Delivery timeline stated with specific dates
- [ ] Warranty terms clearly defined
- [ ] Payment terms 30 days from invoice (government standard)
- [ ] Escalation or price adjustment mechanism if allowed

### 10. Submission Ready Check
- [ ] Tender box or portal confirmed for correct submission method
- [ ] Submission checklist from tender document followed
- [ ] All pages numbered consistent with tender instruction
- [ ] Bid bond or guarantee attached if required
- [ ] Original plus required number of copies prepared
- [ ] Submitted before deadline with minimum 1 hour buffer
- [ ] Submission proof obtained (receipt or confirmation)

## SA Government Tender Evaluation Framework

### 80/20 System (Contracts up to R1,000,000)
- Price (lowest = 80 points): Max 80
- B-BBEE Level: Max 20
- Total: 100

### 90/10 System (Contracts over R1,000,000)
- Price (lowest = 90 points): Max 90
- B-BBEE Level: Max 10
- Total: 100

---

## 5 Common Disqualifiers in SA Tenders
1. Tax clearance not compliant — Check CSD profile before bidding
2. B-BBEE certificate expired or invalid — Set calendar reminders 90 days before expiry
3. Missing mandatory declarations — Print checklist from tender document and tick each
4. Non-compliant pricing format — Use exactly the format specified
5. Late submission — Submit 2 hours before deadline (portals shut precisely on time)
