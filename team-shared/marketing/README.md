## Key Copy Decisions

### Positioning
- **Primary differentiator:** Speed (24h) + Industry specificity + SME focus
- **Tone:** Professional but not corporate. Direct, empathetic to SME pain points
- **Target keywords:** "AI tender writing", "bid proposal service", "capability statement writing", "RFP response service"

### Pricing Naming (ZAR)
- Single Proposal Pack (R750 – R2,500)
- Winning Bid Package (R3,500 – R12,000)
- Tender Accelerator (R5,000 – R20,000/mo)
- Bid Desk (R15,000 – R50,000+)
- Free first bid for early adopters to drive initial traction

### SEO Strategy
- Long-tail keywords targeting each industry (Logistics, Construction, Security, Cleaning, Gov Supply)
- Blog/content topics identified: "How to Write a Winning Tender Response", "Common RFP Mistakes SMEs Make"

### Outreach Channels Ranked
1. **LinkedIn** (organic DM outreach to owners/directors)
2. **Cold Email** (targeted industry sequences)
3. **Industry Facebook Groups** (community engagement)
4. **Personal Network** (warm intros for beta testers)
