# Security Tender Compliance Checklist
## Essential Documents for Winning Security RFPs

### Pre-Submission Checklist
- [ ] Company Security Licence (state-specific)
- [ ] ASIAL Membership (if applicable)
- [ ] Public Liability Insurance ($10M+ min.)
- [ ] Workers' Compensation Insurance
- [ ] Professional Indemnity Insurance
- [ ] Master Security Licence
- [ ] Guard Registration Certificates (all personnel)
- [ ] Monitoring Centre Accreditation
- [ ] ISO 9001:2015 Quality Management
- [ ] ISO 45001:2018 OH&S Management

### Response Must-Haves
- [ ] Executive Summary addressed to evaluation panel
- [ ] Company capability statement (max 2 pages)
- [ ] Guard profiles with certifications
- [ ] Shift coverage plan (24/7 if required)
- [ ] Incident response protocol
- [ ] Technology integration details (CCTV, access control)
- [ ] Training and competency matrix
- [ ] Reporting and compliance documentation
- [ ] Pricing schedule (fixed fee or per-hour)

### Common Disqualifiers to Avoid
- Missing insurance certificates
- Expired licences
- Incomplete pricing schedules
- Vague methodology statements
